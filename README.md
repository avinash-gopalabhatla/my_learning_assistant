# my_learning_assistant
This is built for someone switching domains who can't track what they actually understood vs what they've merely read.

I have used NotebookLM for learning from particular sources and I have always liked the fact that NotebookLM doesn't hallucinate information and the UI itself feels in a way that you are using only selective resources to look for. I like the idea of isolated sources and makes me feel I am in control of the knowledge base and helps me feel whatever answers I get I am getting it based on solely my inputs.

however in several instances I end up creating multiple notebooks and I tend to create a new knowledge base again from scratch. I always wished if I could may be import some other notebook so that I can collectively see what I want rather than upload the whole database into single notebook. Basically I wished if there is a collective database like our brain that has several segregations of information and how they are always tagged in straight forward and in a bizarre way to somehow bring correlate different topics through minute invisible threads which helps us create analogies between things.

I wish to create such tool where the actual knowledge base is single and i bring up whatever is required to certain level of depth based on the context. If I am learning something new I should be able to upload my references and my existing knowledge base should help the tool in giving me the best way of explaing the new topic which i believe is the personlized way of learning.
