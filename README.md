# my_learning_assistant

# I have always felt I have learnt and improved more in the presence of intellectual and understanding friends and I wish to create one such for every ambitious learner

This is built for someone switching domains who can't track what they actually understood vs what they've merely read.

I have used NotebookLM for learning from particular sources and I have always liked the fact that NotebookLM doesn't hallucinate information and the UI itself feels in a way that you are using only selective resources to look for. I like the idea of isolated sources and makes me feel I am in control of the knowledge base and helps me feel whatever answers I get I am getting it based on solely my inputs.

however in several instances I end up creating multiple notebooks and I tend to create a new knowledge base again from scratch. I always wished if I could may be import some other notebook so that I can collectively see what I want rather than upload the whole database into single notebook. Basically I wished if there is a collective database like our brain that has several segregations of information and how they are always tagged in straight-forward and yet in a bizarre way to somehow correlate different topics through minute invisible threads which helps us create analogies between things.

I wish to create such tool where the actual knowledge base is single and i bring up whatever is required to certain level of depth based on the context. If I am learning something new I should be able to upload my references and my existing knowledge base should help the tool in giving me the best way of explaining the new topic which i believe is the personalized way of learning. NotebookLM has several features to generate slide decks, quiz memory map everything, but yet again it is on collective sources and if I wish to see specific to a group of sources I should regenerate them yet again. this involves the initiation from user, I wish to have it in such a way that the tool provides questionnaire to track my understanding. NotebookLM need not have this because it is not claiming to be a learning partner.

I also felt I wasn't able to track whether I understood everything that I added as a source, however simple it might seem to get a summary out of 3 4 sources I believe one has to understand each and every corner of their source to truly see and understand the bigger picture. And that is something I want to add in the tool which helps me be aware of the level of understanding for each and every source so that I would not just have the gist and summary of it rather I would be able to have the working knowledge of it. 

And that's why I am building this tool which has a collective brain that segregates the information you put in a layered chain fashion and determines your understanding on each and every source and suggests you the area you need to focus upon thus giving a personalized learning companion experience.
